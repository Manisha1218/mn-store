# 🛍️ MN Store — Modern React Ecommerce Website

MN Store is a fully responsive modern ecommerce website built using React.js.  
It includes shopping cart functionality, product quantity management, checkout system, and multiple payment methods like UPI, COD, and Card Payment.

---

# 🚀 Features

✅ Modern Responsive UI  
✅ React Functional Components  
✅ Dynamic Shopping Cart  
✅ Increase / Decrease Product Quantity  
✅ Checkout Page  
✅ Payment Gateway UI  
✅ UPI Payment Options  
✅ QR Code Payment  
✅ Card Payment Form  
✅ Cash On Delivery Option  
✅ Multiple Product Categories  
✅ Attractive Product Cards  
✅ Mobile Responsive Design  
✅ Fast and Lightweight
